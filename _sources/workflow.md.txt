# Workflow

A good workflow is:

1. Create or obtain a perfectly symmetrical base mesh, for example right after applying the Mirror modifier.
2. Make sure the mesh is clean, symmetrical, and does not contain duplicated vertices.
3. Create the OmniMirror configuration immediately.
4. Edit the mesh.
5. Use that persistent symmetry configuration later to symmetrize anything you need: the base mesh, shape keys, vertex groups, or UV maps.
