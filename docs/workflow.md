A good workflow is:

1. Create or obtain a perfectly symmetrical base mesh (for example right after applying the Mirror modifier).
2. Ensure the mesh is clean symmetrical and do not contain double/duplicated vertices.
3. Create the OmniMirror configuration immediately.
4. Edit the mesh.
5. Use that persistent symmetry configuration later on to symmetrize whatewer you like: basis mesh, shape keys, vertex groups, UV maps.
